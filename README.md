# Viktor Zouboulis — personal site

Astro + Tailwind personal site with a light terminal / informatics aesthetic.

## Pages

- `/` — home
- `/research` — papers & projects
- `/cv` — curriculum vitae
- `/about` — bio & hobbies
- `/blog` — writing

## Develop

```bash
npm install
npm run dev
```

## Content

- Blog posts: `src/content/blog/`
- Papers: `src/content/papers/`
- Projects: `src/content/projects/`
- CV / About copy: edit `src/pages/cv.astro` and `src/pages/about.astro`
